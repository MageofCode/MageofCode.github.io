The Problem

Let's Get Started

Installation

From Windows to Caliburn in four steps

No USB drives. No command lines. No scary menus. Just a normal installer.

1

Download the installer

Click the download button below. It's a normal Windows .exe file, about 2GB.

Download caliburn-install.exe
2

Run it like any program

Double-click the file. Windows will ask "Do you want to allow this app to make changes?" — click Yes. This is normal.

3

Follow the wizard

Choose your options — which desktop, which apps. The installer handles the rest. Then it tells you to restart.

4

Restart — you're done

Your computer restarts into Caliburn Linux. No Windows required. Everything is set up and ready to use.

Prefer the USB method?

Download the ISO and flash it to a USB drive with Balena Etcher. Boot from USB and the installer will guide you through.

FAQ

Questions people actually ask

Will my files be deleted?

The installer will ask you before touching anything. You can keep your files, or start fresh — your choice.

Can I still use my printer / webcam / headphones?

Most hardware works out of the box. Caliburn ships with drivers for the vast majority of common devices.

What about Microsoft Office?

LibreOffice is included — it opens Word, Excel, and PowerPoint files. For browser-based Office, that still works fine too.

Can I still watch YouTube and Netflix?

Yes. Librewolf (our default browser) handles both. Multimedia codecs are included in the installer.

Is it really free? What's the catch?

No catch. Caliburn is open source software built by volunteers. We don't sell your data because that's the whole point.

What if something goes wrong?

The community forum is the right place for help. Most common questions have answers already waiting for you.

I'm not technical. Can I really do this?

Yes. That's who Caliburn was built for. If you can install a program on Windows, you can install Caliburn.

Does Caliburn phone home or collect data?

Never. No telemetry. No accounts required. No data leaves your computer unless you send it yourself.
